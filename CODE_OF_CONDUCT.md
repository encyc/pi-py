# 贡献者公约

## 我们的承诺

为营造一个开放和友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、民族、性别认同和表达、经验水平、国籍、外表、种族、宗教或性取向如何，参与本项目都不会受到骚扰。

## 我们的准则

有助于创造积极环境的行为包括：

- 使用热情和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注什么对社区最有利
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性关注或性挑逗
- 挑衅、侮辱/贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确允许发布他人的私人信息，如实际地址或电子地址
- 在专业环境中合理地认为不适当的其他行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受行为采取适当和公正的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，或暂时或永久禁止任何他们认为有不当、威胁、冒犯或有害行为的贡献者。

## 适用范围

本行为准则适用于所有项目空间和代表项目或社区的公共空间中的个人。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账户发布或作为指定代表参加在线或离线活动。

## 执行

可以通过 [GitHub Issues](https://github.com/earendil-works/pi-py/issues) 报告辱骂、骚扰或其他不可接受的行为。所有投诉都将被审查和调查，并将产生被认为必要且适合具体情况的回应。维护者有义务对事件报告者保密。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.0。
